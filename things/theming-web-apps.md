# Theming web apps (and browsers)

Want to write themes for web apps? Fuck off [here](#writing-web-app-themes).

## Discord

Use [Vencord](https://vencord.dev) (or [Vesktop](https://github.com/Vencord/Vesktop) for a more lightweight client with Vencord preinstalled). As an alternative, [Equicord](https://equicord.org) is a fork of Vencord with more plugins.

## Firefox

For custom CSS see [this tutorial](https://www.reddit.com/r/FirefoxCSS/wiki/index/tutorials/), and for custom JS see https://github.com/MrOtherGuy/fx-autoconfig

## Spotify

Use [Spicetify](https://spicetify.app).

## Steam

Use [Millennium](https://steambrew.app). Note that themes have the ability to use custom JS, so you won't have to make your own plugin to do something simple.

## Visual Studio Code

For colors you may look up the colorscheme you want in "Extensions" (open with Ctrl+Shift+X).

For custom CSS use the [Custom UI Style](https://github.com/subframe7536/vscode-custom-ui-style) extension. Currently, [Windows 95](https://github.com/ricewind012/win95-themes/blob/master/docs/vscode.md) is the only one of its kind, so you can look at its code to see how it's done.

## Vivaldi

See [this guide](https://forum.vivaldi.net/topic/10549/modding-vivaldi) featuring both custom CSS and JS.
